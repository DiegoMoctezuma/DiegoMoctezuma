## 🚀 Proyectos destacados

### 🤗 Clasificación de Imágenes Sintéticas
Pipeline distribuido con Apache Spark para procesamiento de 88k+ imágenes y extracción de características. Entrenamiento de modelos de clasificación utilizando Spark MLlib y EfficientNet-B0 con transfer learning, alcanzando un AUC de 0.97.  
Modelo desplegado en Hugging Face Spaces para inferencia en tiempo real.

---

### 🛒 FrontendProyect — Plataforma para la Comunidad Estudiantil
Desarrollo de una plataforma web para la gestión y venta de productos nuevos y de segunda mano dentro de la comunidad estudiantil de la FES Acatlán. El proyecto también contempla publicación de servicios y otras interacciones entre estudiantes.

Desarrollado en equipo bajo el nombre **4deAsada** como parte de la materia de Desarrollo Web.

---

### 📈 FinanSaber — Glosario Financiero Inteligente
Aplicación móvil desarrollada con React Native y Expo enfocada en facilitar el aprendizaje de términos financieros mediante una experiencia interactiva y accesible.
La app permite explorar un glosario dinámico, buscar conceptos específicos y utilizar un agente integrado para resolver dudas financieras de forma conversacional.

---

### 🏛️👁️ ARGOS — Human-Centered AI para Accesibilidad Educativa
Proyecto enfocado en reducir las barreras académicas para estudiantes con discapacidad visual mediante visión computacional, modelos de lenguaje y asistentes de voz. La aplicación permite analizar contenido de pizarrones y materiales educativos usando OCR y modelos de IA para generar resúmenes, explicaciones y cuestionarios interactivos accesibles mediante voz.

El proyecto incorpora principios de **Human-Centered AI (HCAI)**, incluyendo transparencia de modelos, indicadores de confianza, control del usuario sobre decisiones de la IA y gestión de privacidad de datos.

---
